# Проект для инвентаризации

## Команды для запуска и управления
```shell
# сборка
make build

# запуск
make run

# создание автоматический миграций
make makemigrations

# создание пустой миграции
make emptymigration

# применить миграции 
make migrate

# откатить последнюю миграцию
make unmigrate
```
